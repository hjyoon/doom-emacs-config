# Doom Emacs Configuration

Custom configuration for Doom Emacs.

## Apply Configuration

1. Clone the repo to `~/.config/doom`:
    ```bash
    git clone https://github.com/hjyoon/doom-emacs-config.git ~/.config/doom
    ```

2. Sync Doom Emacs:
    ```bash
    doom sync
    ```

3. Restart Emacs.
